### 重载

Java/C++ 中是有函数重载的(即相同函数名，但传的参数不同是当成不同函数的),而 JavaScript 没有函数重载，旧其本质是因为 JavaScrit 中函数是一个对象。函数名类似一个指针。

### label 语句的用法

区分以下两段函数：

```js
let num = 0
for (let i = 0; i < 10; i++) {
  for (let j = 0; j < 10; j++) {
    if (i === 5 && j === 5) {
      break
    }
    num++
  }
}
console.log(num) // 95
```

使用 label 语句：

```js
let num = 0
outPoint:
for (let i = 0; i < 10; i++) {
  for (let j = 0; j < 10; j++) {
    if (i === 5 && j === 5) {
      break outPoint
    }
    num++
  }
}
console.log(num) // 55
```
